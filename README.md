## 大模型作业



​		面向谷子规模化种植基地中出苗监测效率低、播期决策依赖经验、精准管理水平不足等问题，融合无人机遥感、深度学习与气象大数据技术，构建了谷子苗情智能监测与最佳播种期决策模型体系。基于中尺度冠层RGB影像，研发了YOLO-EFC-OBB谷子出苗率检测模型，引入增强特征相关性模块与定向边界框检测机制，针对幼苗“小尺度、密集分布、易重叠、背景复杂”等特性进行优化，在独立测试集上实现93%以上的检测准确率，实现单株级自动识别、出苗率精确计算及缺苗区域空间定位，并生成补苗处方图。

1. mizhi-pestcontrol 为后端服务程序文件
2. plus-ui 是前端服务程序
3. 两个压缩包是对应的flask 模型接口
